# ADR-005: Type Mapping Policy

- **状态**: 已接受
- **日期**: 2026-05-15 (初版) / 2026-05-20 (更新)
- **提出者**: 架构决策

## 背景

sqlc 解析 SQL 查询后报告 PostgreSQL 列类型（int4、text、timestamptz 等）。Codegen 需要将这些类型映射到 MoonBit 类型。

## 决策

### 映射表

| PG 类型 | MoonBit 类型 | 备注 |
|---------|-------------|------|
| `int2`, `smallint`, `smallserial` | `Int` | 16-bit 小整数 |
| `int4`, `integer`, `int`, `serial` | `Int` | 32-bit 平台字宽 |
| `int8`, `bigint`, `bigserial` | `Int64` | 64-bit 需要 |
| `float4`, `real` | `Double` | |
| `float8`, `double`, `double precision` | `Double` | |
| `numeric`, `decimal` | `String` | 保留完整精度 |
| `bool`, `boolean` | `Bool` | |
| `text`, `varchar`, `char`, `bpchar`, `name`, `character varying` | `String` | |
| `bytea` | `Bytes` | |
| `json`, `jsonb` | `JsonValue` | |
| `uuid`, `inet`, `cidr`, `macaddr` | `String` | MVP fallback |
| `date` | `Date` | runtime 封装 |
| `timestamp`, `timestamptz`, `timestamp with time zone` | `DateTime` | runtime 封装 |
| `time`, `timetz` | `String` | 无封装 |
| `interval` | `String` | MVP fallback |
| 未知类型 | `String` | 安全 fallback |

### Getter 映射

| MoonBit 类型 | Row getter | Nullable 变体 |
|-------------|-----------|--------------|
| `Int` / `Int64` | `get_int64(pos)` | `get_nullable_int64(pos)` |
| `String` | `get_string(pos)` | `get_nullable_string(pos)` |
| `Bool` | `get_bool(pos)` | `get_nullable_bool(pos)` |
| `Double` | `get_double(pos)` | `get_nullable_double(pos)` |
| `Bytes` | `get_bytes(pos)` | `get_nullable_bytes(pos)` |
| `JsonValue` | `get_json(pos)` | `get_nullable_json(pos)` |
| `Date` | `get_date(pos)` | `get_nullable_date(pos)` |
| `DateTime` | `get_datetime(pos)` | `get_nullable_datetime(pos)` |

### Value 构造器

| PG 类型 | Value 变体 |
|---------|-----------|
| 所有整数类型 | `Int64(value: Int64)` |
| `float4`, `float8`, `real`, `double` | `Double(value: Double)` |
| `numeric`, `decimal` | `String(value: String)` |
| `bool`, `boolean` | `Bool(value: Bool)` |
| `text`, `varchar`, ... | `String(value: String)` |
| `bytea` | `Bytes(value: Array[Byte])` |
| `json`, `jsonb` | `JsonValue(value: Json)` |
| `date` | `Date(value: Date)` |
| `timestamp`, `timestamptz` | `DateTime(value: DateTime)` |

### 数组类型

PG 数组通过 `Arr(inner)` 包装，映射为 `Array[T]`：

```moonbit
Arr(Named("int4")) → Array[Int]
Arr(Arr(Named("text"))) → Array[Array[String]]
```

### Nullable 包装

可为空列映射为 `Option[T]`。

## 理由

- **安全 fallback**：未知类型映射为 `String` 保证生成代码始终可编译
- **渐进增强**：映射表可在 ADR 更新时扩展，无需修改 codegen 逻辑
- **数组递归**：通过 IR 的 `Arr` 类型自然处理多维数组
- **精度保留**：numeric/decimal 映射 String 避免 Double 精度丢失
- **类型安全**：date→Date、timestamp→DateTime 提供编译期类型检查

## 后果

- 正向：映射表集中管理，修改映射只需改 `type_map.mbt`
- 正向：未知类型不导致编译失败
- 正向：时间/日期类型提供专用封装，保留语义
- 正向：numeric/decimal 无精度丢失
- 负向：time/timetz、interval 仍为 String，需后续扩展
- 负向：Value 枚举变体随类型扩展增加维护成本

## 替代方案

- **编译期报错**：未知类型导致失败，过于严格
- **自定义类型映射**：允许用户配置映射文件，超出 MVP 范围
- **所有类型→String**：过于宽松，丢失类型安全优势
